## üìå Overview

**AgriBankMS** is an **academic and prototype-level Bank of Agriculture Management System** designed to model and demonstrate **digital banking workflows** commonly found in **agricultural finance institutions**.

The system focuses on supporting:
- Agricultural loan processing  
- Farmer and cooperative account management  
- Credit monitoring and repayment tracking  
- Administrative and reporting operations  

This project is intended for **educational, research, and demonstration purposes**.
